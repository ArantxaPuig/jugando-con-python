# 🐍 Jugando con Python

Repositorio creado para practicar y mejorar habilidades en Python, enfocado en fundamentos, estructuras de datos, lógica y resolución de problemas.

---

## 📌 Descripción

Este repositorio contiene ejercicios, scripts y pequeños proyectos desarrollados durante el proceso de aprendizaje de Python.

El objetivo es consolidar conocimientos mediante la práctica constante y la resolución de problemas reales.

---

## 🎯 Objetivos

- Mejorar la lógica de programación
- Dominar la sintaxis de Python
- Trabajar con estructuras de datos
- Aprender buenas prácticas de código
- Preparación para proyectos de análisis de datos

---

## 🧠 Contenidos trabajados

- Variables y tipos de datos
- Condicionales (`if`, `elif`, `else`)
- Bucles (`for`, `while`)
- Funciones
- Listas, tuplas, diccionarios y sets
- Comprensiones de listas
- Manejo básico de errores
- Entrada y salida de datos
